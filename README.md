# Shopify Announcement App

A Shopify app that lets store owners set a storefront announcement banner from the Admin panel.

## Tech Stack
- React + Polaris (Admin UI)
- Remix (Full-stack framework)
- Node.js + Express
- MongoDB (Audit history)
- Shopify Metafields API
- Theme App Extension (Storefront banner)

## Setup

### Prerequisites
- Node.js v18+
- MongoDB Atlas account
- Shopify Partner account

### Installation

```bash
git clone https://github.com/yourusername/announcement-app.git
cd announcement-app
npm install
```
